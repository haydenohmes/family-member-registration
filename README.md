# Family Member Registration

A Next.js application for family member registration, implementing the design from Figma.

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000/registration](http://localhost:3000/registration) in your browser.

## Project Structure

- `/app/registration` - Main registration page
- `/components/ui` - Reusable UI components (Button, Avatar)
- `/lib` - Utility functions

## Technologies

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Radix UI
- Lucide React (icons)
- Uniform Web Design Tokens
